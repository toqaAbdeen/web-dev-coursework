HTML & CSS Fundamentals - Beginner's Guide

Getting Started with HTML & CSS

What's It About?

This article covers the fundamental building blocks of web development - HTML and CSS. Learn how to create your first webpage and style it.

Category: HTML & CSS Basics

Article Content

HTML (HyperText Markup Language)

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

Example HTML structure:

<!DOCTYPE html>

<html>

<head>

  <title>My First Page</title>

  <link rel="stylesheet" href="style.css">

</head>

<body>

  <h1>Welcome</h1>

  <p>This is my first webpage!</p>

</body>

</html>

CSS (Cascading Style Sheets)

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Example CSS structure:

body {   background-color: lightblue; }

h1 {   color: white;   text-align: center; }

p {   font-family: verdana;   font-size: 20px; }

To begin:

  1. Create a file named index.html
  2. Add basic HTML structure with doctype, html, head, and body tags
  3. Insert content using paragraph (p), heading (h1-h6), and other HTML elements
  4. Create a separate style.css file
  5. Link the CSS file in your HTML head section
  6. Start styling elements with CSS selectors and properties


JavaScript DOM Manipulation

JavaScript DOM Manipulation Basics

What's It About?

Learn how to dynamically interact with and modify webpage content using JavaScript and the Document Object Model (DOM).

Category: JavaScript & Frontend

Article Content

What is JavaScript?

JavaScript is the programming language of the web.
It can update and change both HTML and CSS.
It can calculate, manipulate and validate data.

Why Study JavaScript?

JavaScript is one of the 3 languages all web developers must learn:

  1. HTML to define the content of web pages
  2. CSS to specify the layout of web pages
  3. JavaScript to program the behavior of web pages
Source: W3Schools JavaScript Tutorial

Key concepts:

  • Selecting Elements: Use methods like getElementById(), querySelector(), etc.
  • Modifying Content: Change text with innerHTML or textContent
  • Styling Elements: Access the style property of DOM elements
  • Event Handling: Respond to user interactions with addEventListener()

Example code:

// Select an element const header = document.querySelector('h1'); // Change its text header.textContent = 'New Heading'; // Change its color header.style.color = 'blue'; // Add click event header.addEventListener('click', function() { alert('Header clicked!'); });

Remember: Always place your script tags at the end of the body or use DOMContentLoaded event to ensure elements exist before manipulating them.
